# __NAME__

A Fluxlay wallpaper.

> Examples below use `npm`. Substitute `bun`, `pnpm`, or `yarn` as you prefer.

## Develop

```sh
npm install
npm run dev
```

The Fluxlay desktop app must be running. It will hot-reload as you edit files.

## Build

```sh
npm run build
```

Produces `wallpaper.fluxlay`.

## Publish

```sh
npx fluxlay login   # one-time
npm run publish
```
